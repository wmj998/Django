# 部署



## uWSGI

+ 安装

+ 配置（uwsgi.ini）

  ```python
  [uwsgi]
  http=127.0.0.1:8000
  chdir=project_dir
  wsgi-file=project_name/wsgi.py
  # 进程
  process=4
  # 线程
  threads=2
  # 服务的pid记录文件
  pidfile=uwsgi.pid
  # 服务的日志文件
  daemonize=uwsgi.log
  # 主进程管理模式
  master=True
  ```

+ 配置（settings.py）

  ```python
  DEBUG=False
  ALLOWED_HOSTS=['网站域名'/'服务监听的ip地址']
  ```

+ uWSGI 运行管理（uwsgi.ini 文件所在目录）

  ```
  uwsgi --ini uwsgi.ini  # 启动
  uwsgi --stop uwsgi.pid  # 停止 
  ps aux|grep uwsgi
  ```
